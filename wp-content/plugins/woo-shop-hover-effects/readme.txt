== Description ==

Makes your Woo Store Attractive and Amazing with 70+ CSS3 Hover Effects and Store on Load Animations
<h3><a href="https://wordpress.org/plugins/woo-shop-hover-effects/screenshots/">Demo</a></h3>

<h3>Features</h3>
<ul>
<li>Compatible with Every Theme</li>
<li>Responsive</li>
<li>25 on Load Animations for WooCommerce Shop</li>
<li>42 Hover Effects for Shop Items and Products</li>
<li>Simple to Use</li>
<li>Light Weight</li>
<li>Based on CSS3 KeyFrames</li>
<li>Custom Animation Duration</li>
<li>Custom Animation Delay</li>
</ul>
== Installation ==

1. Go to plugins in your dashboard and select 'add new'
2. Search for 'Woo Shop Hover Effects' and install it
3. Go to WooCommerce > Hover Effects, and configure settings
4. Now visit store
